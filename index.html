What am I looking at?

Every structure in this scene is grown from one real tree record in NYC's 2015 Street Tree Census, fetched live from NYC Open Data. Species, trunk diameter, health, and location all come from the real record — nothing is invented except the branch shape itself, which is procedurally grown (seeded per tree, so the same record always grows the same shape).

Reading the map

  • Outlines are the official NYC borough boundaries — water is already excluded, so gaps between shapes are rivers/harbor, not missing data.
  • The blue underneath is a rendered water base, not photography — it's there so land reads as solid against the harbor and rivers.
  • The dashed outlines are New Jersey (west) and Long Island's Nassau/Suffolk counties (east) — geographic context only, neither has tree data of its own. Long Island's real coastline extends past the default view; pan or zoom out to see the North and South Forks.
  • Canopy color — Health: green = Good, ochre = Fair, red = Poor, grey = Unknown.
  • Canopy color — Species: each species gets a consistent hue; the legend below the map shows the top species in the current sample.
  • Branch diameter and depth scale with the tree's real trunk diameter (DBH) — bigger, older trees grow taller, more branched structures. Branch geometry only renders once you're zoomed in close (from far away, thousands of overlapping branches just read as a muddy blob, so canopy dots alone carry the view).
  • The scale bar and borough labels are derived from the real lat/long projection, not estimated.

Controls

  • Orbit: click/touch and drag.
  • Pan: right-click drag, shift+drag, or two-finger drag.
  • Zoom: scroll wheel or pinch.
  • Recenter: snaps the camera back to the current view preset, undoing any pan/orbit/zoom.
  • View — 3D / Plan: switch between an oblique angle and a true top-down map view.
  • Compass dial (top-right of the map): click to reset orientation to north-up without changing zoom or pan.
  • Tap a tree: any canopy point can be tapped/clicked to show that tree's real record — species, trunk diameter, health, borough, and tree ID.
